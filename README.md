CreditCardEditText
==================

Single credit card input view for android apps
(based on https://github.com/zdfs/toscani/)

<img src="https://raw.githubusercontent.com/julioz/CreditCardEditText/master/creditcardedittext.gif"/>

Minimum API required: API 15 (Ice Cream Sandwich 4.0.3)

### How to use
To use CreditCardEditText in your project, you just need to add this project as a library project to your workspace and reference it in your project.
